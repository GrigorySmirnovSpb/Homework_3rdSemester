# Movie database analysis app
This is an application for analyzing a movie database. It processes data such as: Movie title, IMDB id, movie rating, names of directors and actors, tags, etc.

Dotnet version 10 is required to run.

How to run:
git clone https://github.com/GrigorySmirnovSpb/Homework_3rdSemester
